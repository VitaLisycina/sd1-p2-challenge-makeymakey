# Function design document MakeyMakey
In this file, all the info you need for your challenge assignment is included.

## Functional requirements
* There should be an interaction between a project on the screen and the person testing your project.
* This interaction should
  * Have an influence on the Unity project.
  * Be a physical interaction that is not just pressing buttons.

## Non-functional requirements
* The physical prototype should be the main focus of this assignment (NOT THE GAME).
